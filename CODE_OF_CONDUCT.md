# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][covenant], a
widely used code of conduct for open source communities. Its full text
is available at:

- <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- Translations: <https://www.contributor-covenant.org/translations/>

## Our pledge

Maintainers and contributors pledge to make participation in this
project a respectful experience for everyone, regardless of personal
background or identity, and to act in ways that contribute to an open
and welcoming community.

## Scope

This Code of Conduct applies in all project spaces — the GitHub
repository (`ducrot/twigcn` and the read-only subsplit
`ducrot/twigcn-bundle`), issue trackers, pull requests, discussions —
and when an individual is officially representing the project in public
spaces.

## Reporting

Reports of unacceptable behavior can be sent privately to the project
maintainer at **<christian@ducrot.de>**. All reports will be reviewed
and investigated promptly and fairly. The maintainer is obligated to
respect the privacy and security of the reporter.

For security-related issues that are not interpersonal, see
[`SECURITY.md`](./SECURITY.md) instead.

## Enforcement

Maintainers will follow the [Contributor Covenant Enforcement Guidelines][guidelines]
when determining the consequences for any action they deem in violation
of this Code of Conduct.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
[guidelines]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines
